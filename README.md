# 🌦️ Advanced Weather Forecast App

A modern weather forecast web application built using **Python Flask** that displays current weather and **5-day forecast** using the OpenWeather API.

---

## 📸 Screenshot

![Project Screenshot](screenshot.png)

---

## 🚀 Features

- Add multiple cities
- Delete saved cities
- Current weather details
- 5-Day weather forecast
- Temperature in Celsius
- Weather condition icons
- Beautiful responsive UI
- SQLite database storage

---

## 🛠️ Technologies Used

- Python
- Flask
- Flask SQLAlchemy
- HTML5
- CSS3
- Bootstrap
- SQLite
- OpenWeather API

---
